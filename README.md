# UGC Ad Prompt System – Styling App

## 📌 Project Overview
This project creates high-converting UGC ad scripts using AI.

## 📱 Product
Styling App (helps users choose outfits and style clothes using what they already have)

## 🎯 Target Audience
Young people who feel confused about what to wear and lack styling confidence.

## 😟 Problem
Many people have clothes but don’t know how to style or pair them properly.

## 💡 Solution
UGC-style ad scripts that feel real and show how the app helps users style outfits easily.

## 🤖 Tools Used
- ChatGPT

## 📦 Output Includes
- Hooks
- Ad Scripts
- CTAs
- Captions

## 🚀 Result
Created relatable and ready-to-use Instagram ad scripts that businesses can directly use.

## 🎬 How to Use
- Record in selfie mode
- Speak naturally
- Use real emotions
- Keep video 20–30 seconds
